
=========
Playtools
=========

Playtools is a technology framework for creating applications for role-playing
games. It encompasses a data format based on N3, tools for editing that
format, an object mapper, and documentation for integrating data into your
game software.

See http://goonmill.org/playtools/

Bug tracker at http://launchpad.net/playtools/

Installation
============

If you are installing with easy_install, you should be aware of an additional
installation step, pt-system-install.  This builds a database which Playtools
uses heavily.

$ sudo easy_install playtools
$ sudo pt-system-install
